# Project Brief: CrossFit Gym Manager

## Project Goal
Создать комплексное Android-приложение для управления CrossFit залом, которое позволяет:
- Атлетам трекировать тренировки и прогресс
- Тренерам создавать WOD и отслеживать результаты групп
- Администраторам управлять пользователями, группами и всей инфраструктурой зала

## Core Requirements

### Роли и права
1. **Атлет**
    - Просмотр WOD своей группы
    - Запись результатов тренировок
    - Просмотр лидерборда (своя группа + общий)
    - Личный PR трекер
    - Просмотр базы упражнений

2. **Тренер**
    - Всё, что может атлет
    - Создание/редактирование WOD для своих групп
    - Просмотр результатов атлетов своих групп
    - Подтверждение рекордов
    - Управление составом группы
    - Добавление упражнений в базу

3. **Администратор**
    - Всё, что может тренер
    - Полное управление пользователями (создание, блокировка, смена роли)
    - Управление группами (создание, удаление, назначение тренера)
    - Полный доступ ко всем WOD и результатам
    - Управление базой упражнений (CRUD)

### Функциональные требования

**Управление группами**
- Группы: "Morning Crew", "Evening Warriors", "Competition Team" и т.д.
- У группы: название, тренер, список атлетов, расписание тренировок
- Атлет может быть только в одной группе (для упрощения MVP)
- Тренер может вести несколько групп

**База упражнений**
- Упражнение: название, категория (Gymnastics, Weightlifting, Cardio, Monostructural)
- Описание, техника выполнения (текст + фото/видео ссылка)
- Единицы измерения: время, вес, повторения, дистанция, калории
- Сложность (beginner/intermediate/advanced)

**База WOD**
- WOD содержит: название, описание, блок упражнений
- Формат: AMRAP, EMOM, For Time, Tabata, Ladder
- Время работы/отдыха (для EMOM/Tabata)
- Целевая группа (можно назначить конкретной группе или всем)
- Дата проведения (можно запланировать на будущее)

### Non-functional Requirements
- Android 7.0+ (API 24)
- Offline-first: Room + синхронизация через Supabase Realtime (WebSocket)
- Row Level Security (RLS) в Supabase для ролей
- Производительность: JOIN-запросы оптимизированы индексами

## Success Metrics
- 80% WOD имеют заполненные результаты в течение 1 часа после тренировки
- Тренеры тратят < 5 минут на создание недельного расписания
- 100% упражнений имеют описание и видео
